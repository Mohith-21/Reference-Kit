Dotnet Installation- BAM TOOL

Quick Steps without pictures:

  1. Login to stager usomawvdntst01.test.intranet


  2. Navigate to D:\Inetpub\AdminScripts\BAM\BAM-2.5.0.0 and open as administrator


  3. Click on stage web option




  4. Scroll to Type of ASPX, fill out site name: csattest1, scroll to TEST1 for environment, Scroll to 4.0.30319 for Framework. (Application pool can be left blank or fill in, it doesn’t matter)
      Click the browse button to navigate to the Source file and the version number will be automatically filled in once you do that.




  5. Now progress window gets displayed .wait till you get the message “task completed “in the details. Then select “ok”




  6. Now choose deploy web option from the BAM tool for ASMX/ASPX or choose Deploy Service option foe WinSvcs/Tibco services.


  7. Chose the appropriate package from the application deployment package and then enter the server names (from the inventory) to the target servers field and select “ADD”.




  8. Now select “deploy”


  9. That's it Installation is done and Follow the same procedure for other components also
