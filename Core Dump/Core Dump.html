Core Dumps on Ecom Servers

What is a CORE dump?

A core file is generated when an application program abnormally terminates due to bug, operating system security protection schema, or program simply try to write beyond the area of memory it has allocated, and so on.

Generating CORE dump

[qcquot@lxomavtecom00 ~]$ ulimit -c unlimited

CORE DUMP will be created on current dir on servers.
