Procedures to take Heap Dump :

Heap Dump generation for an Application pointing to JDK and JROCKIT:

If Application is pointing to JDK then following STEPS:

  1. export PATH=/prod/appl/bea/home10.3/jdk1.6.0_17/bin:$PATH

  2. cd /data/ecom2/bea//log

    3. (eg : cd /data/ecom2/bea/qcquotingDomain/log)

  3. jmap -J-d64 -dump:format=b,file=./'Name_of_Dump_file_to be_Created'.hprof -pid-

    4. (eg : jmap -J-d64 -dump:format=b,file=./MS1_300.hprof 19312 )

If Application is pointing to JROCKIT then following STEPS:

EG: RXPS TEST1

  1. export PATH=/prod/appl/bea/home10.0.1/jrockit_150_65/bin:$PATH

  2. cd /prod/ecom2/local/apps/apps10/rxps/current/logs

  3. jrcmd pid hprofdump filename=./logs/rxps2.hprof

    4. or Run using below command

    /prod/appl/bea/home10.0.1/jrockit_150_65/bin/jrcmd 13141 hprofdump filename=./logs/rxps2.hprof
