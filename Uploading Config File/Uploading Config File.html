Uploading Config File

How to Perform a Configuration Update with a New Config File

  1. Open up the request to see what file is being requested to be updated.




  2. Download the Config file from attachments or,
    3. https://centurylink.sharepoint.com/sites/SPTETS-Team/IETS/ETSLibrary/Forms/AllItems.aspx

  3. Login to WinScp through JumpServer




  4. Copy downloaded file from local system to JumpServer using Drag and Drop in WinScp.




  5. Spawn the servers using Spawn Utility and login.


  6. In Admin server go to the given path in the instruction and take the backup of existing file.


  7. Using SCP command get the file from JumpServer to current location


    8. Ex: scp -r ad34087@vlodjumpts00.dev.qintra.com:/tmp/ad34087/US_export_policy.jar .

  8. In stager using WLSDEPLOY script sync all the manageed servers with option 12




  9. Bounce the server using 7 and 8 Options




  10. Check the URLs and close the ticket.
